# Spring AI Learning Journey 🚀

A structured, hands-on journey building AI-powered applications using Spring Boot and Spring AI with a locally hosted LLM via Ollama.

This project demonstrates how to integrate a Java backend with a free, locally running Large Language Model (LLM) without relying on paid cloud APIs.

---

## 🧠 Tech Stack

- Java 17+
- Spring Boot 3
- Spring AI
- Ollama (Local LLM Runtime)
- Llama 3 Model
- Maven
- Bootstrap 5 (Frontend UI)

---

## 🖥️ Requirements

Before running this project, make sure you have:

- Java 17 or higher
- Maven 3.9+
- macOS / Linux / Windows
- Minimum 8GB RAM (16GB recommended for better performance)
- Internet connection (for first-time model download)

---

## 🦙 Install Ollama (Free Local LLM)

This project uses the Llama 3 model running locally via Ollama.
